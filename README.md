# Systematic Trading Pipeline

End-to-end quantitative trading research system built in Python.

## Pipeline

Signal → Position → PnL → Risk Normalization → Returns → Equity Curve

## Features

- Signal generation framework
- Position sizing with bounded exposure
- Return normalization for stability
- Equity curve + drawdown analysis
- Modular architecture

## Stack

Python, NumPy, Pandas, Matplotlib

## Structure

- src/ → core logic
- notebooks/ → research
- visuals/ → dashboards
